# Setup

# 1. Software Installation

  • Conda Package Management Systemopen in new window
PDAL Installation in Conda
conda create --name pdal
conda activate pdal
conda install -c conda-forge pdal python-pdal gdal

Information

Nach Ende des Workshops lässt sich das Environment mit folgendem Befehl entfernen:

conda env remove --name pdal
  • QGIS Version >= 3.28open in new window

Ein Texteditor wie z. B. nanoopen in new window oder Visual Studio Codeopen in new window wird für die Bearbeitung der JSONopen in new window-Dateien benötigt.
