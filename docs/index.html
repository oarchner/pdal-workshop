# Zusammenfassung

Die Point Data Abstraction Bibliothekopen in new window ist eine sehr leistungsfähige Bibliothek zur Verarbeitung von Point Cloud Daten. In unserem Workshop lernen sie alle wichtigen Elemente der Bibliothek kennen. Detailliert gehen wir auf die umfangreichen und sehr mächtigen Filterfunktionen zum Beispiel zur Detektion von Bodenpunkten ein. Bei allen Beispielen rufen wir PDAL von der Kommandozeile aus auf und verwenden QGIS als Visualisierungswerkzeug. Ein Komplexbeispiel am Ende des Workshops zeigt viele der vorgestellten Funktionen im praktischen Kontext und gibt ihnen eine gute Vorlage für den Einsatz von PDAL in eigenen Projekten.
